# Brass Template — Deepak Products

## Project Overview
A reusable B2B landing-page template for budget brass/industrial manufacturers, built with React + Vite. Deepak Products (Jamnagar, est. 1985) is the first instance.

## Goals
- Beat ishitabrass.com (current #1 for "brass components") on every dimension: design, UX, lead capture, content depth
- WhatsApp-first lead capture (no backend required)
- Rebrand a new client by editing `client-config.js` + CSS vars only

## Stack
- React + Vite (static output)
- CSS custom properties (no CSS-in-JS, no Tailwind)
- React Router v6 for inner pages
- react-helmet-async for SEO
- Netlify deploy (`vite build` → `app/dist/`)

## Key Constraint
No paid libraries. Google Fonts + Lucide icons (CDN) only.
